# Test directory

## 1 Basic Structure
```
test/
    src/
    tests/
    run_tests   <-- Main executable
```

- The `src` directory contains the base `tfc_TestSystem`, its config file
  `TestSystemCONFIG.yaml` 
  ```
  src/
      RELAP_custom_src/
      tfc_TestSystem/
      TestSystemCONFIG.yaml
  ```
- The `tests` directory contains the tests.
